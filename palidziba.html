Kas ir Vārddienis WEB?

Vārddienis WEB pamatā ir īpaši veidota interneta saite (aplikācija), kas pilda ierastās Vārddieņa funkcijas. Aplikācija savā darbībā izmanto jaunākos HTM5 un JavaScript risinājumus, proti, jQuery un jQueryMobile, kas to padara daudz pievilcīgāku un ērtāk lietojamu ikvienam datora lietotājam.

Šis projekts tika izveidots, lai ļautu Vārddieni izmantot arī mobilajās ierīcēs (Android, Symbian, iPhone, J2ME, ...), kā arī Linux un Mac datoros, kam iepriekš šāda iespēja nebija pieejama.

Saderīgās ierīces un pārlūkprogrammas

Vārddieni WEB var izmantot lielākajā daļā ierīču un pārlūkprogrammu (mobilajos telefonos, datoros, planšetdatoros u.c.), kas atbalsta jaunāko HTML un JavaScript standartu darbību. Jāpiebilst, ka daži dizaina elementi un animācijas ierīcēs var tikt attēloti daļēji vai arī var netikt attēloti, tādēļ apstiprināts saraksts ar atbalstītajām platformām un pārlūkprogrammām apskatāms šeit: http://jquerymobile.com/gbs.

Pazīstamāko pārlūkprogrammu atbalsts:
  • Chrome Desktop 11+ (visaugstākā elementu un animāciju attēlošanas kvalitāte);
  • Firefox Desktop 4+;
  • Opera Desktop 10+;
  • Internet Explorer 7+;
  • Opera Mobile 11.0+.

Lietošanas noteikumi un projekta pirmkods

Vārddienis WEB ir un būs bezmaksas projekts, kura attīstību veicinās gan Jūsu sniegtie ziedojumi, gan izteiktie komentāri, viedokļi un ierosinājumi, gan mana paša vēlme to pilnveidot un uzlabot, un izmainīt.

Tā kā šis projekts ikvienam tiek piedāvāts bezmaksas, tad arī to ir aizliegts jebkādā veidā komerciāli pārdot, izplatīt vai pavairot, jo īpaši bez jebkādas saskaņošanas ar autoru. Ikviens interesents var izmantot piedāvāto pirmkodu (https://bitbucket.org/arvislacis/varddienis_web/overview), lai veidotu līdzīgus bezmaksas izstrādājumus, sekotu līdzi uzlabojumiem un izmaiņām projekta pirmkodā.

Lūgums Jūsu izstrādājumā, ja tādu izveidosiet, tomēr ietvert kādas norādes uz šo saiti, autoru, kā arī par savu veikumu man pastāstīt vairāk.

Ko iesākt, ja esat pamanījis kļūdu?

Šo projektu pagaidām izstrādāju tikai es (Arvis Lācis), protams, ka reizi pa reizītei arī es, kā jau cilvēks, kļūdos vai arī nepamanu kādu mazu niansi skriptā, tādēļ aicinu ikvienu, kurš Vārddienī WEB pamanījis kādu kļūdu vai nepilnību; arī tad, ja ir radusies vēlme ko ieteikt vai vienkārši padiskutitēt, sazināties ar mani personīgi un izteikt savu viedokli.
